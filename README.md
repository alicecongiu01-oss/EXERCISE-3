This project implements a Multi-Layer Perceptron (MLP) to generate realistic Sardinian names. 
The model is trained on a dataset of 991 traditional Sardinian names collected from Wikipedia.
The dataset is included in the repository: [https://github.com/alicecongiu01-oss/EXERCISE-3/blob/main/Abele.txt].
The model learns to predict the next character in a sequence based on a fixed number of previous characters, following the approach introduced by Bengio et al. (2003) and the implementation template provided by Andrej Karpathy.
For comparison, a character-level bigram model is also provided as a baseline, highlighting qualitative differences in the names generated by the two models.
